# World Clock
A clock for people who would like to see the time in various time zones across the globe.

## Usage
```
$ mongod
$ npm install
$ npm run seed
$ npm start
```

## Uses
- HTML5
- CSS3
- Javascript(ES2015+)
- moment.js
- moment-timezone.js
- MongoDB
- Express
- Node.js
